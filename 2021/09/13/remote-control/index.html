Remote Control

2021-09-13

Description

For this project, I created an application that allows the user to connect their client to a second remote computer. This is useful for a variety of situations. For example, are there times when you forget your file in another computer? Do you want to access two computers at one time by just using a small program to connect?
Here is what you want.

def deal_data(conn, addr):
  print (‘Accept new connection from {0}’.format(addr))
  # conn.settimeout(500)
  # reply when get the request
  conn.send(“Hi, Welcome to the Server”.encode(“utf-8”))
  while 1:
    fileinfo_size = struct.calcsize(‘128sl’)
    # receive the file name
    buf = conn.recv(fileinfo_size)
    # whether receive the heading message
    if buf:
      # get the filename and size
      filename, filesize = struct.unpack(‘128sl’, buf)
      fn = filename.strip(b’\00’)
      fn = fn.decode()
      print (‘file new name is {0}, filesize is {1}’.format(str(fn),filesize))

      recvd_size = 0
      # save the file to the same folder as this program
      fp = open(‘./‘ + str(fn), ‘wb’)
      print (‘start receiving…’)

      # write the file
      while not recvd_size == filesize:
        if filesize - recvd_size > 1024:
          data = conn.recv(1024)
          recvd_size += len(data)
        else:
          data = conn.recv(filesize - recvd_size)
          recvd_size = filesize
        fp.write(data)
      fp.close()
      print (‘end receive…’)
    break
